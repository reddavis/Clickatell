= Clickatell

This is a wrapper for http://www.clickatell.com 

== Install

  gem sources -a -http://gemcutter.org
  sudo gem install reddavis-clickatell
  
== How To Use

Sending Texts

  require 'rubygems'
  require 'clickatell'
  
  a = Clickatell::Text.new('username', 'password', 'api_id')
  a.send(number, 'this is the message')
